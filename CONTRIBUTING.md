# Contributing to BSM FiveM AutoBackup

First off, thank you for considering contributing to this repository! It's people like you that make the open source community such a great place to learn, inspire, and create.

This script is maintained by **Basil Saji Mathew (BSM)**.

### How Can I Contribute?

1. **Bug Reports:** 
   Feel free to submit an issue outlining any bug you find. Be sure to include your OS, Node Version, and the exact error from the FiveM console.
   
2. **Feature Requests:**
   Have a great idea that will make this script even better? Create an issue outlining the feature request and why it'd be beneficial.

3. **Pull Requests:**
   - Fork the Project
   - Create your Feature Branch (`git checkout -b feature/AmazingFeature`)
   - Commit your Changes (`git commit -m 'Add some AmazingFeature'`)
   - Push to the Branch (`git push origin feature/AmazingFeature`)
   - Open a Pull Request

### Code Style Guidelines
- Please keep code clean and heavily commented, similar to the existing BSM code base.
- Do not remove the primary author credits/watermarks (Basil Saji Mathew (BSM)) from the main files.

Happy Coding!
